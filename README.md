DESCRIPTION: <br/>
Created a recipe finder to make it easier to find cook and find recipes that match the ingredients you have in your fridge. In this application, you can input the different ingredients you have in your fridge, and using the Spoonacular API, it will traverse through all recipes, and find only those that match the ingredients you inputted. All recipes will have a link to view all ingredients needed and a link to view the recipe. <br/>

TECHNOLOGIES USED: <br/>
Frontend: HTML/CSS, JavaScript <br/>
API: Spoonacular API
